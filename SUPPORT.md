# Support

## Where to Ask for Help

- Usage questions and troubleshooting: open a GitHub issue with context.
- Bug reports: open a GitHub issue with a minimal reproducible example.
- Security issues: follow `SECURITY.md` and report privately.

## What to Include

Please include:

- Rust version (`rustc --version`)
- OS and architecture
- Enabled crate features
- Catalog JSON / provider config (redact secrets)
- Full error messages and logs
- Minimal code sample to reproduce

## Issue Types

Use clear titles and labels where possible:

- `bug`
- `question`
- `documentation`
- `enhancement`

## Response Expectations

This is an OSS project and response time can vary.
Well-scoped reports with reproducible steps are handled faster.
